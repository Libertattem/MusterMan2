**********MUSTERMAN**************

Задача данного приложения упростить работу в поверке счетчиков воды.
MusterMan программа для поверителей, служит в роли органайзера с подведением итогов за смену или за все время.
В далгосрочной перспективе, использовать ее в комерчиских целях.

Идеи механик для программы:

    1. Получение маршрута с сервера и/или возможность добавлять адреса в ручную (реал. 30%)

    ****** основной раздел маршрута **********
    2. Меню обработки адреса (количество работ/финансы) (реал. 80%)
    3. Меню денежных движений за смену (реал. 60%)

    ********** итоги **********
    4. Подведение итогов за смену (реал. 50%)
    5. Раздел общей статистики (реал. 10%)
    6. Раздел профиля работника (реал. 10%)

    ******* настройки *********
    7. Раздел настроек профиля (реал. 10%)
    8. Раздел настроек программы (реал. 10%)
    9. Раздел настроек маршрута (реал. 10%)

    ******* прочее *******
    10. Меню общей навигации (реал. 50%)
    11. Чаты, рабочие или между сотрудниками. Внутренний чат компании (реал. 0%)
    12. БД для хранения основной(общей) статистики (реал. 0%)
    13. БД для хранения информации по СИ* (ГРСИ и пр.) (реал. 0%)
    14. Интеграция с сервером (реал. 0%)
    15. Формирование итогов в виде электронного документа (PDF) (реал. 0%)
    16. Формирования маршрута на гугл карте (реал. 0%)


*СИ - средсва измерений (счетчики воды)
